# Mj vs Lebron Analysis

Using quarto, and a hanful of python libraries, I was able to compare and contrast the careers of Michael Jordan and Lebron James.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Quarto Software
2. Install HTML Software
3. Download the included Python libaries

Instructions for using the software:

1. Select the preview button in VS Code
2. Create a rendered document with Quarto
3. Use a localhost on browsers

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Windows	quarto-1.9.37-win.msi
* Python 3.13
* Polars
